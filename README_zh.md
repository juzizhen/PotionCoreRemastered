# ✨ 药水核心重制版

> 为游戏增加了许多药水效果。

![Header Image](https://i.ibb.co/q351946C/mcmod-icon.png)

---

## 📖 简介
- 部分灵感和部分贴图来源于 [PotionCore](https://github.com/Tmtravlr/PotionCore)，但没有使用其开源代码。
- 在 PotionCore 的基础上进行了改进。
- 本模组不自带合成（酿造）配方，需要的可以自行使用 [KubeJS](https://github.com/KubeJS-Mods/KubeJS) 自定义配方。
- 本仓库开源许可为 **GPL-3.0**。
- 遇到任何问题请务必到[**issues**](https://github.com/juzizhen/PotionCoreRemastered/issues)提交问题！

---

## 🧪 效果一览

### ✅ 正面效果
|    名称    | 效果描述                                                                              | 作用对象  |
|:--------:|:----------------------------------------------------------------------------------|:-----:|
|  **解毒**  | 自动清除身上的中毒效果并免疫中毒。                                                                 |  实体   |
| **提高精度** | 每级增加 **3** 点投掷物伤害。                                                                |  实体   |
|  **祝福**  | 每Tick获得 **效果等级 x 1** 种时间为 1分钟 的正面效果。                                              |  实体   |
|  **炸裂**  | 释放强度为 **效果等级 * 2** 的冲击对实体造成击退效果和伤害，不会对玩家造成伤害，不会破坏地形。                              |  实体   |
|  **攀爬**  | 对着墙向前走可以像蜘蛛一样爬墙。                                                                  |  玩家   |
|  **净化**  | 每Tick清除所有负面效果。                                                                    |  实体   |
|  **镶钻**  | 每级增加 **4** 点盔甲韧性。                                                                 |  实体   |
|  **延长**  | 该buff等级≥其它状态效果数量，则其它状态效果的持续时间不会降低，直至该buff消失。<br/>若该buff等级<其它状态效果数量，则其它状态效果轮流延长时间。 |  实体   |
|  **修复**  | 每隔5秒，恢复主手/副手/四个装备栏各 **效果等级** 点耐久度。<br/>注：效果消失CD不会重置！                              |  玩家   |
|  **复活**  | 获得效果后免疫一次死亡，在生效后，此效果会消失，并回复 **效果等级×4** 点的生命值。<br/>注：部分特殊伤害无法防御，如秒杀圈神器。            |  玩家   |
|  **高踏**  | 每级增加 **0.5** 点踏步高度。踏步高度：无需跳跃即可走上的高度。                                              |  实体   |
|  **回家**  | 拥有该药水效果在不移动的情况下10秒后会被传送至重生点（可以跨维度传送），传送过程中会产生大量粒子，移动即取消。                          |  玩家   |
| **凋零抗性** | 自动清除身上的凋零效果并免疫凋零。                                                                 |  实体   |
|  **稳固**  | 每级增加 **100%** 的免疫击退效果。                                                            |  实体   |
| **魔力聚焦** | 每级增加 **75%** 的药水伤害。注：中毒药水无效！                                                      |  实体   |
|  **飞行**  | 创造飞行同款。                                                                           |  玩家   |
|  **爱情**  | 使动物进入求爱模式，效果与喂食动物一致，不会使动物无视冷却强制繁殖。注：该效果无法使玩家获得爱情！                                 | 非玩家实体 |
| **地面传送** | 获得此效果时，无论处于何处，都会被传送到最近的地表。                                                        |  实体   |
|  **延伸**  | 增加拥有该药水效果的实体的触及半径和攻击距离（效果等级×1格 ）。                                                 |  玩家   |
|  **铁皮**  | 每级增加 **2** 点护甲值。                                                                  |  实体   |
|  **魔抗**  | 每级增加 **4** 点魔法护盾。在药水效果结束前续杯可以叠加药水时间，魔法护盾用法见下方 **[特殊玩法](#特殊玩法)** 描述。               |  玩家   |
|  **反伤**  | 有该效果会在受伤时对伤害源造成 **受到伤害 x 效果等级 x 25%** 点反伤伤害，反伤不会再次被反伤，反伤伤害属于魔法伤害。                 |  实体   |
|  **有序**  | 拥有该效果会定期自动整理背包物品，不包括物品栏、盔甲栏和主副手，默认周期 **30** 秒，可在配置文件调整。                           |  玩家   |

### ❌ 负面效果
|    名称    | 效果描述                                                                     | 作用对象 |
|:--------:|:-------------------------------------------------------------------------|:----:|
| **护甲减效** | 每级减少 **2.5** 点护甲值。                                                       |  实体  |
|  **诅咒**  | 每Tick获得 **效果等级 x 1** 种时间为 1分钟 的负面效果。                                     |  实体  |
|  **乱序**  | 每Tick随机打乱物品栏和背包的物品。                                                      |  玩家  |
|  **驱散**  | 每Tick清除所有正面效果。                                                           |  实体  |
|  **鱼鳃**  | 水下可以正常呼吸，但是在陆地上呼吸时会消耗自身氧气槽。                                              |  实体  |
|  **爆破**  | 每Tick产生强度为 **效果等级 x 2** 的爆炸，会对玩家造成伤害，会破坏地形。                              |  实体  |
|  **燃烧**  | 每Tick使其进入 **效果等级 x 10** 秒着火状态。                                           |  实体  |
| **精度下降** | 每级减少 **4** 点投掷物伤害。                                                       |  实体  |
|  **脆弱**  | 每级增加 **50%** 受到伤害。                                                       |  实体  |
|  **腐蚀**  | 每秒减去身上所有装备 **效果等级** 点耐久。对主手和副手上的物品同样有效。                                  |  实体  |
|  **镜像**  | 该效果会使你做出相反的操作效果。效果：<br/>向左移动<->向右移动；向前移动<->向后移动；抬头<->低头；左转<->右转；潜行<->跳跃。 |  玩家  |
| **魔力抑制** | 每级减少 **30%** 的药水伤害。注：中毒药水无效！                                             |  实体  |
|  **破魔**  | 每级减少 **魔法护盾总量10%** 点魔法护盾。魔法护盾用法见下方 **[特殊玩法](#特殊玩法)** 描述。                 |  玩家  |
|  **头晕**  | 每Tick使玩家的视角向随机方向偏移 **效果等级 x 2~10** 点角度。                                  |  玩家  |
|  **沉重**  | 每级减少 **5%** 移速和 **3%** 跳跃高度。                                             |  实体  |
| **随机传送** | 每5Tick尝试一次传送，当传送位置安全时进行传送。                                               |  实体  |
|  **闪电**  | 每 **5 - 效果等级** 秒会在所在的位置生成一道闪电。                                           |  实体  |
|  **发射**  | 效果等级为 **Ⅰ** 时击飞高度为 **6** 格，之后每增加一级击飞高度为 **效果等级 x 6 + 2** 格。              |  实体  |
| **药水病**  | 该效果没有药水形式。用法见下方 **[特殊玩法](#特殊玩法)** 描述。                                    |  玩家  |
|  **禁空**  | 禁止玩家创造飞行和鞘翅飞行，禁止除末影龙外的其他生物飞行。                                            |  实体  |
|  **枯竭**  | 每级减少 **4** 点最大生命值。                                                       |  实体  |

|:------:|:-----------------------------------------------------------|:----:|

### 特殊玩法
|  名称  |                            描述                             |
|:----:|:---------------------------------------------------------:|
| 魔法护盾 | 魔法护盾可以减免相应数值的魔法伤害（包括反伤），在魔抗效果期间获得的额外魔抗都将叠加魔法护盾值，直到魔抗效果消失。 |
| 药水病  | 当正面buff过多时，会受到随机debuff，默认不启用，可在配置文件里选择启用，也可在配置文件里设置相应的概率。 |
